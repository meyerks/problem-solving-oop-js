# OOP Scheduling & Resource Allocation (JavaScript)

This repository contains a collection of Object-Oriented Programming (OOP) exercises in JavaScript, focused on solving real-world system design problems.

The tasks simulate interview-style scenarios such as:
- flight and gate scheduling
- resource allocation
- constraint handling
- simple system design logic

Each task includes:
- problem description
- predefined classes (as in real coding tests)
- clean and readable solution

## Goal

- Practice OOP in realistic scenarios
- Improve ability to work with existing codebases
- Prepare for technical interviews (HackerRank-style tasks)
